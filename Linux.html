Linux

¿Qué es Linux?

Linux es un sistema operativo de código abierto que permite controlar el hardware de una computadora y ejecutar programas Fue creado en 1991 por Linus Torvalds y actualmente es desarrollado por miles de programadores alrededor del mundo.

¿Qué es GNU/Linux?

Es el nombre completo y técnicamente correcto del sistema operativo. Se llama así porque combina el Kernel Linux (el motor) con las herramientas y programas del proyecto GNU (el cuerpo del sistema, como el compilador, la terminal y las utilidades básicas). Sin GNU, Linux no sería un sistema funcional para el usuario.

¿Por qué Linux no es un Sistema Operativo?

Linux, por sí solo, es únicamente un componente (el núcleo). Un sistema operativo completo requiere una interfaz de usuario, un sistema de ventanas, gestores de archivos y aplicaciones. Linux es la pieza central, pero no es el todo; es como el motor de un coche: el motor es esencial, pero no es "el coche" completo hasta que le pones chasis, ruedas y asientos.

¿Qué es un Kernel?

El Kernel o núcleo es el corazón de cualquier sistema operativo. Es el software de nivel más bajo que actúa como puente invisible entre el hardware físico (procesador, memoria, periféricos) y las aplicaciones que tú usas (navegador, juegos, editores de texto).

Datos curisos

Tux, la Mascota: El pingüino es el logo oficial desde 1996. Linus Torvalds eligió un pingüino porque le parecen animales simpáticos (y cuenta la leyenda que uno lo mordió en un zoológico).

Open Source (Código Abierto): A diferencia de Windows o macOS, cualquiera puede leer el código de Linux, modificarlo y compartirlo. Esto lo hace muy seguro porque miles de personas lo revisan constantemente.

¿Sabías que Android es Linux? Mucha gente no lo sabe, pero el sistema operativo de miles de millones de teléfonos usa el Kernel de Linux. Aunque la interfaz y las apps son distintas a las de una PC, el "motor" que mueve tu celular es el mismo del que estás hablando en tu proyecto.

¿Dónde está Linux? ¡En todos lados! No solo en PCs. El 100% de las 500 supercomputadoras más potentes del mundo usan Linux, y también está en tu teléfono Android, en servidores web e incluso en autos inteligentes.
